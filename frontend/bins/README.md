# [Bins](https://caquita.app) Frontend Webapp

[Deno](https://deno.com) + [Vue 3](https://vuejs.org) + [TresJS](https://tresjs.org) (ThreeJS wrapper)

## Development

```bash
deno install
```

```bash
deno run dev
```

#### Cleaning:

```bash
deno clean
```

#### Building:

```bash
deno run build
```
